# 75 Days Challenge Dashboard

## Project Overview

This project is an interactive Excel dashboard created to track and analyze performance during a 75-day self-improvement challenge.

The dashboard transforms daily habit-tracking data into meaningful insights through KPIs, charts, and performance metrics.

---

## Objectives

* Monitor daily habit completion
* Track consistency over time
* Measure overall challenge performance
* Identify strongest habits
* Visualize progress trends

---

## Tools Used

* Microsoft Excel
* Pivot Tables
* Pivot Charts
* KPI Cards
* Excel Formulas
* Dashboard Design & Data Visualization

---

## Dashboard Features

### KPI Cards

* Current Day
* Consistent Days
* Average Daily Score
* Completion Percentage
* Best Habit
* Performance Status
* Progress Status

### Visualizations

* Daily Performance Trend
* Weekly Average Score Trend
* Habits Completion Comparison
* Habits Completion Distribution
* Score Distribution (Doughnut Chart)

### Insights Section

The dashboard automatically summarizes challenge performance using calculated metrics.
